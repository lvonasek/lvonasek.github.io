3D Live Scanner - Android storage bug workaround


There's a new bug since Android 11. This bug prevents apps compiled with the latest SDK from accessing public folders. The problem occurs when the app is reinstalled or when a third-party app writes something to the folder. Google Play forces developers to use the latest SDK.

Workaround

Step 1. Uninstall the current version of 3D Live Scanner.

Step 2. Locate 3D Live Scanner data folder. Note that screenshots show the Samsung file manager, but the step is very similar in any file manager.


Step 3. Rename folder to "Models"


Step 4. Install an older version of the 3D Live Scanner: Donwload APK

Step 5. Update the app using Google Play Store

When the application is restarted, the folder migration process starts and the Android system grants access rights to all data.
